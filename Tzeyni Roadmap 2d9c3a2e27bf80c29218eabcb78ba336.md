# Tzeyni Roadmap

### Feature Implementation Roadmap

### Phase 1: MVP (Months 1-3)

**Core Features:**

- User authentication (clients/professionals)
- Professional profile creation
- Service listing and management
- Basic booking system
- Payment (mazal khasna nfkro fih)
- Review and rating system

**Technical Implementation:**

1. Set up Next.js frontend with authentication
2. Create Rails API with core models
3. Implement professional onboarding flow
4. Build booking system with calendar integration

### Phase 2: Enhanced Features (Months 4-6)

**Additional Features:**

- Advanced search and filtering
- Real-time messaging system
- Professional dashboard analytics
- Automated notifications (SMS/Email)
- Multi-language support (Arabic/French)
- Professional verification system

**Technical Implementation:**

1. Add WebSocket support for real-time features
2. Implement geolocation services
3. Build comprehensive dashboard
4. Add notification infrastructure
5. Implement multi-language support
6. Add professional verification workflow

### Phase 3: Growth Features (Months 7-10)

**Advanced Features:**

- Professional community features
- Advanced analytics and reporting
- Loyalty programs
- Professional training resources
- Admin panel for platform management

**Technical Implementation:**

1. Build community features
2. Add advanced analytics
3. Implement loyalty system
4. Create admin dashboard
5. Add performance monitoring

### Monetization Strategy

### Revenue Streams:

1. **Commission Model:** 10% / 15% commission on completed bookings (lower than Fresha's 20%)
2. **Premium Subscriptions:** Enhanced features for professionals
    - Basic: Free (limited bookings per month)
    - Pro: $19.99/month (unlimited bookings + analytics)
    - Premium: $39.99/month (marketing tools + priority listing)
3. **Payment Processing Fees:** 2.9% + $0.30 per transaction (hta nchoufo CMI ola chi haja khra)
4. **Advertising:** Promoted professional listings
